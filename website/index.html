Core value

Handle GitHub triage, reviews, and documentation without waiting for your laptop. NovaFlux builds authenticated requests and drafts responses directly from your device.

Why it matters

  • Stay responsive on the move with Codex-quality drafts.
  • Share PR checklists and issue templates with one tap.
  • Liquid-glass UI keeps the experience bright, legible, and tactile.

How to use (2 steps)

  1. Open GitHubStarter.playground in Swift Playgrounds or Xcode and tap Run.
  2. Tap Trigger NovaFlux automation to send a Codex-backed request, then share or copy the response.

Need visuals? The 2-minute script walks through the exact beats.

API setup

Swap in your live endpoint and key to run real automations. Defaults are safe for demos.

NOVFLUX_API_BASE = https://api.novaflux.example/v1
NOVFLUX_API_KEY = nova-demo-key-please-replace

Update these values in your environment before running the playground to perform live requests.
